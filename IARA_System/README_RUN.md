# Как да стартирате ИАРА приложението

## Стъпки за стартиране:

### 1. Стартиране на API сървъра
Отворете терминал в папката `IARA_System/IARA.API` и изпълнете:
```bash
dotnet run
```
API-то ще стартира на: **http://localhost:5028**

### 2. Стартиране на Web приложението
Отворете **НОВ** терминал в папката `IARA_System/IARA.Web` и изпълнете:
```bash
dotnet run
```
Web приложението ще стартира на: **http://localhost:5164** (или друг порт)

### 3. Вход в системата
- Отворете браузъра и отидете на адреса на Web приложението
- Използвайте тестовите данни:
  - **Потребител:** admin
  - **Парола:** Admin123!

## Важни бележки:

1. **API трябва да работи преди Web приложението** - Web приложението се свързва с API-то
2. **Базата данни** - При първо стартиране, API-то автоматично създава базата данни и добавя примерни данни
3. **CORS** - Вече е конфигуриран в API-то за да позволява заявки от Web приложението

## Тестови данни в системата:

След първото стартиране, системата автоматично създава:
- 3 рибаря
- 2 кораба
- 2 разрешителни (едното изтича след 15 дни - за тест на Справка 1)
- 1 билет за любителски риболов
- Няколко улова и записи в дневника

## Достъпни страници:

- `/` - Начална страница (Табло)
- `/ships` - Регистър на корабите
- `/licenses` - Разрешителни за риболов
- `/reports` - Справки и анализи (4 справки)
- `/add-catch` - Регистрация на нов улов
- `/login` - Вход в системата

## Swagger API документация:

След стартиране на API-то, можете да видите документацията на:
**http://localhost:5028** (Swagger UI)
